Техническое задание
Разработать программу, позволяющую шифровать и расшифровывать текст с использованием шифра Цезаря.

Программа должна поддерживать несколько режимов работы:
1. шифровка текста;
2. расшифровка текста с помощью ключа;
3. расшифровка текста с помощью brute force;
4. выход

Структура программы:

- MainApp: Главный класс, управляющий работой программы.
- FileManager: Обрабатывает чтение и запись файлов.
- Cipher: Содержит методы шифрования и расшифровки текста.
- Validator: Проверяет допустимость входных данных.
- BruteForce: Реализует метод брутфорса для расшифровки текста.
